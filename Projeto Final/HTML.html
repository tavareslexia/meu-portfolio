HTML5: A Estrutura Semântica da Web

Bem-vindo à seção de aprendizado em HTML. Aqui você encontrará recursos e informações sobre HTML.

O que é HTML5?

HTML5 é a quinta versão da linguagem de marcação HTML, que é usada para estruturar e apresentar conteúdo na web.

Elementos Estruturais:

<div>

  • Descrição: Elemento de contêiner genérico usado para agrupar outros elementos.
  • Atributos: id, class, style
  • Quando usar: Para estruturar páginas, dividir seções e aplicar estilos com CSS.

<p>

  • Descrição: Define um parágrafo de texto.
  • Atributos: class, id, style, title
  • Quando usar: Para exibir blocos de texto

<a>

  • Descrição: Cria um hiperlink para outra página, site, ou parte do mesmo documento.
  • Atributos: href, target, rel, title
  • Quando usar: Sempre que quiser criar links clicáveis.

<section>

  • Descrição: Representa uma seção temática de um documento.
  • Atributos: id, class
  • Quando usar: Para dividir o conteúdo em blocos lógicos, como “Sobre nós”, “Serviços”, “Contato”.

<article>

  • Descrição: Conteúdo independente, como uma notícia, post de blog ou comentário.
  • Atributos: id, class
  • Quando usar: Quando o conteúdo pode ser distribuído de forma independente ou reutilizável.

<header>

  • Descrição: Cabeçalho de uma página ou seção.
  • Atributos: id, class
  • Quando usar: Para agrupar títulos, logotipos, menus ou resumos no topo da página/seção.

<footer>

  • Rodapé da página ou seção
  • Atributos: id, class
  • Quando usar: Para incluir informações como copyright, links ou contato.

<nav>

  • Descrição: Representa um grupo de links de navegação.
  • Atributos: id, class, aria-label
  • Quando usar: Para menus principais, barras laterais ou rodapés com navegação.

<main>

  • Descrição: Conteúdo principal da página, único por documento.
  • Atributos: id, class
  • Quando usar: Para envolver o conteúdo central, excluindo cabeçalho, barra lateral e rodapé.

<aside>

  • Descrição: Conteúdo complementar, como barras laterais, blocos de sugestões ou publicidade.
  • Atributos: id, class
  • Quando usar: Quando o conteúdo está relacionado mas não é parte do fluxo principal.
